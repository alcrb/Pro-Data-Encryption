```bash
----------------------------------------------------------------------
OPTIONS
```

The _encrypt file_ is a **CLI** for encrypting and decrypting files.

## Requirements
This project is tested with:

| Requisite      | Version  |
|----------------|----------|
| Python         | 3.11.0   |

## Install
```sh
pip install encrypt-file
```


- :unlock: Decrypt :unlock:

```bash
          File Name or Path File you need to Decrypt :
          Final File Name :
```
